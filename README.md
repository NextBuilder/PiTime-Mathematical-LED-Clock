# 🧠 **Project Overview**

Have you ever looked at a clock and wished it felt a little more interesting? PiTime transforms a simple clock into a small thinking experience by replacing numbers with mathematical expressions. Instead of reading the time instantly, you pause for a moment and work it out.

The display uses a clean LED-based design to show time in a simple and intuitive way, all inside a compact, 3D-printed enclosure. Powered by a rechargeable battery, it stays completely portable and wire-free. It shows time in a creative way, encourages you to pause and think, and adds a modern, unique touch to your desk or study space.

It’s a simple idea brought to life by combining math, electronics, and design into one unique build.


# 🕒 **How It Works**

The concept is simple and intuitive:

1. The clock has **12 LEDs** arranged in a circle
2. Each LED represents an hour position
3. Time is shown using colors:

   * 🔴 **Red → Hour**
   * 🟢 **Green → Minute (5-minute steps)**
   * 🔵 **Blue → When both overlap**

Example:
**3:25 → LED 3 = Red, LED 5 = Green**

# 🧩 **Components Used**

| Component                    | Description           |
| ---------------------------- | --------------------- |
| **Raspberry Pi Pico 2**      | Main controller       |
| **PCF8563 RTC Module**       | Accurate time keeping |
| **WS2812B RGB LEDs (12x)**   | Time display          |
| **18650 Battery**            | Portable power        |
| **Charging Module (Custom)** | Battery charging      |
| **3D Printed Enclosure**     | Clean circular design |
